## 三、实验总结及体会

* 安装GLUT时文件拷贝的位置一定要正确；可能会因为Code Blocks的安装位置不同导致文件拷贝的位置不同；
* 编写程序时将#include <windows.h>放在#include <GL/glut.h>后面导致程序报错，后将#include <windows.h>放在最前面成功解决此问题。
